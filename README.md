Life Management System – Java Edition

Life Management System je Java desktop aplikacija koja pomaže korisnicima da prate i organizuju svoje svakodnevne aktivnosti, navike, zadatke, san i finansije. Aplikacija omogućava jednostavan pregled ličnih podataka, praćenje napretka i analitiku kroz četiri glavna modula: Habit Tracker, Sleep Tracker, Study Planner i Finance App.

Funkcionalnosti

- Login/Registracija korisnika uz pohranu lozinke i tema.
- Personalizacija teme: zelena, plava, roza, narandžasta, tamna (dark) i cyberpunk.
- CRUD operacije za svaki modul (dodavanje, brisanje, označavanje završenog).
- Analitika: osnovna statistika i sažeti prikazi za svaki modul.
- Export podataka u TXT fajl za sve module.

 Moduli

1.Habit Tracker – prati dnevne navike i njihov završetak.
2.Sleep Tracker – evidentira sate sna i nudi osnovnu analitiku.
3.Study Planner – prati zadatke i njihov napredak.
4.Finance App – bilježi prihode i rashode, prikazuje stanje i rashode po kategorijama.

Tehnologije

Java 
Swing (GUI)
MongoDB (baza podataka)

Pokretanje

1. Pokrenuti MongoDB lokalno.
2. Clone-ati repozitorij i otvoriti u IntelliJ IDEA ili drugom IDE-u.
3. Pokrenuti `Main.java` za start aplikacije.
